# Hospital-Emergency-Room

Hospital Emergency Room Dashboard

📖 Description

Interactive Excel Dashboard for analyzing Emergency Room patient data, wait times, admission status, patient satisfaction, age groups, and department referrals.

🎯 Objectives
Monitor patient flow
Analyze wait times
Track admission rates
Measure patient satisfaction
Analyze department referrals

🛠️ Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting

📊 KPIs
Total Patients
Average Wait Time
Patient Satisfaction Score
Admission Rate
Gender Distribution

📈 Dashboard Insights
Age-wise patient distribution
Gender-wise analysis
Admission vs Not Admitted analysis
Department referral analysis
Monthly trend analysis

🚀 Outcome

Developed an interactive dashboard that provides meaningful insights into emergency room operations and supports data-driven decision making.

🔗 Skills Demonstrated
Data Analysis
Data Cleaning
Dashboard Development
Data Visualization
Excel Reporting
